# THOMPRESSOR™ v1.0

**Vintage FET Limiter • Handcrafted in Helsinki**

Retro analog compressor PWA by Tomi "Thom Force" Forsblom.

## Features
- Authentic 1176-style FET compression
- Real-time VU meter (Input / Output switchable)
- 6 touch-friendly knobs (Threshold, Ratio, Attack, Release, Makeup, Tone)
- Reactive tube glow + interference flicker
- Input source selector (Demo / Mic / Line)
- Built-in VHS-style recorder (REC / PLAY / SAVE / DEL)
- Fully offline PWA (installable to home screen)
- 60 fps smooth animations

## How to use
1. Open in any modern browser (best on iOS Safari or Chrome)
2. Tap **POWER** to activate
3. Select input source (Demo = internal test signal, Mic = live microphone)
4. Drag knobs up/down to adjust
5. Double-tap any knob to reset
6. Use **REC** in the VHS deck to record (microphone + compressor output)
7. Save recordings as .webm files

## Installation (PWA)
- Open the link on your phone
- Tap **Share** → **Add to Home Screen**
- You now have a real app icon with offline support

## Technical
- Pure vanilla HTML/CSS/JS + Web Audio API
- No external dependencies
- Follows THOMPRESSOR-AGENTS.md guidelines
- Mobile-first touch support
